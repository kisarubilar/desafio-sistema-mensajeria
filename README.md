# Desafío 1 - Sistema de Mensajería 💬

Este proyecto consiste en la maquetación responsiva de la interfaz de una aplicación de mensajería y redes de contacto para la empresa **Contact Me SPA**. El objetivo principal fue estructurar el layout utilizando **CSS Flexbox** de forma precisa, limpia y eficiente.

## 🛠️ Requerimientos Implementados

- **Estructura Principal:** Layout de dos columnas principales (Sección de contactos y Sección principal) utilizando contenedores flexibles.
- **Sección de Contactos (Sidebar):** - Estado del usuario con foto, nombre y estatus con opacidades sutiles y distribución horizontal (`space-between`).
  - Botón "Nuevo Chat" centrado y estilizado.
  - Lista de contactos ordenada verticalmente con detalles de último mensaje e imágenes alineadas con Flexbox.
- **Sección Principal:** Contenido de bienvenida centrado de forma vertical y horizontal, con optimización del tamaño del logo y tipografía para asegurar la jerarquía visual.

## 🧰 Tecnologías Utilizadas

- **HTML5:** Estructura semántica y limpia.
- **CSS3:** Estilo global y uso intensivo de **Flexbox** para el diseño responsivo.
- **Git & GitHub:** Control de versiones y despliegue.
